# AluHub Aluminium Fabrication Website

Internal website for AluHub Aluminium Fabrication with staff salary advance management system.

## Features

1. **Homepage**
   - Professional design with aluminium works gallery
   - Services showcase
   - Responsive layout

2. **Admin Portal**
   - Staff salary advance management
   - Add/edit/delete advance records
   - Staff-wise summary
   - Search functionality
   - Data export to CSV

## Admin Access
- **Password**: AluHub@123
- **Admin Page**: `admin.html`

## Setup Instructions

1. **Create a GitHub Repository**
   - Go to GitHub.com
   - Create new repository named `aluhub-website`
   - Make it public (for free hosting)

2. **Upload Files**
   - Upload all 5 files to your repository
   - Create an `images` folder for aluminium work photos

3. **Enable GitHub Pages**
   - Go to Repository Settings
   - Scroll to "Pages" section
   - Select "main" branch as source
   - Click Save

4. **Access Your Website**
   - Your site will be available at: `https://[your-username].github.io/aluhub-website`

## Data Storage
- All data is stored in browser's localStorage
- Data persists per device/browser
- No server required

## Customization

1. **Colors**: Modify the color variables in `style.css`
2. **Images**: Replace Unsplash URLs with your actual aluminium work photos
3. **Password**: Change `ADMIN_PASSWORD` in `script.js`

## Notes
- This is for internal use only
- Data is stored locally on each device
- Regular backups recommended (export CSV)
